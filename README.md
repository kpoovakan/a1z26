# A1Z26 Enciphering And Deciphering

Quickly encipher and decipher messages using the A1Z26 cipher.
In this simple monoalphabetic cipher, A=01, B=02, C=03, and so on,
all the way to Z=26. Use or preview the program
<a href="https://kpoovakan.github.io/a1z26">here</a>.
